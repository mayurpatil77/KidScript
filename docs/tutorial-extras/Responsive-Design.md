### Media Queries: The Secret Code

üìö Responsive design uses a special trick called media queries. It's like telling your web page, "Hey, if you're on a small screen, do this!" With media queries, you can change how things look and work on different devices.


**Media Query Example:**

```css
/* <!-- This is a media query --> */
@media (max-width: 600px) {

 /* Styles inside this block will apply only when the screen width is 600px or less */

  body {
    background-color: lightblue;
  }
}
```
